# Georgian Alphabet Trainer

A web app for learning the Georgian alphabet by gradually mixing Georgian letters into English/Russian text.

## Features

- **Script Mixing**: Passive learning through context.
- **Interactive Controls**: Adjustable difficulty.
- **Custom Content**: Practice with your own text.
- **Modern UI**: Clean, accessible, and runs entirely in the browser.

## Quick Start

```bash
# Install
git clone https://github.com/cneltyn-s/georgian-alphabet-trainer.git
cd georgian-alphabet-trainer
yarn install

# Run
yarn dev
```
